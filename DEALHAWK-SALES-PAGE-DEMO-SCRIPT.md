## FAQ block (anticipated objections)

> **Will this template do all the work for me?** No. The desk does the sourcing, the scoring, the underwriting math, the script drafting, and the disposition packaging. You make the decisions: which leads to pursue, when to upgrade modes, when to engage an attorney, when to accept an offer. The agents are tools. You're the operator.

> **Do I need an attorney for every deal?** No. Standard wholesale assignments don't require one — your title company handles the closing paperwork. Attorney review is **strongly recommended** for Sub-To and creative-finance structures (novation, wraps, lease-options, contract-for-deed) and in statute-heavy states (IL, OK, NJ, NY, CA, MA, MD, VA, TN, PA, SC). When an attorney is on file for the property's state, the desk cites them in the paperwork. The compliance checklist surfaces this on your dashboard daily.

> **What about TCPA / DNC / state wholesaler-disclosure rules?** Built in. Outreach mode requires you to attest to TCPA compliance before it unlocks. Every SMS auto-appends the STOP opt-out footer. Every disposition output auto-inserts the state-specific wholesaler disclosure (50 states + DC matrix). The output sanitizer catches prohibited promise language before it leaves the system. The opt-out velocity check on your compliance dashboard warns if you're getting too many opt-outs in a 7-day window.

> **What data provider do I need?** None to start. The Demo provider returns realistic synthetic leads so you can exercise the entire flow before subscribing to anything. When you're ready to go live, the BatchData adapter is built in — drop your API key and the same agents work against real data. PropStream / REsimpli / Tracerfy adapters can be added on the same provider interface.

> **What if I'm in a strict-disclosure state?** The desk handles it. Strict-state deals (IL, OK, SC, MA, NY, NJ, etc.) get a banner on every disposition recommending double-close instead of assignment. The address fuzzes to general-area only in marketing copy. The state-specific disclosure block is auto-inserted. You still need to verify with a state-licensed attorney, but the paperwork doesn't ship with the wrong language by accident.

> **What's the catch?** No catch. You don't need this template to work a hundred times. You need it to work once.

## Voice direction (for ElevenLabs / Claude voice generation)

- **Tone:** Confident, direct, operator-grade. Not salesy. Not hype. Read it like you'd read it to a peer who already knows real estate.
- **Pace:** Slightly faster than conversational. Aim for 165–175 wpm.
- **Inflection:** Drop pitch on the dramatic numbers ("ninety-six thousand dollars at zero down"). Pause before verdict text.
- **Avoid:** Up-talk. Vocal fry. "Imagine this." "What if I told you." Generic ad-voice cadence.
- **Reference voice:** Patrick Bet-David in his shorter videos. Or the Hard Money Bankers podcast. Operator-to-operator tone.
