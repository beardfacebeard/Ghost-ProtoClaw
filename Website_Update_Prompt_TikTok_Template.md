## Step 3: Visual Design Direction

The featured section should feel like a premium upgrade moment on the page. Here's the design direction:

**Color / Glow Treatment:**
- Ghost Operator card: Use a ghostly purple/violet gradient glow or border animation — fits the "ghost" branding
- TikTok Shop Operator card: Use a TikTok-inspired gradient (cyan → magenta / #00f2ea → #ff0050) for the glow or border — immediately signals "this is TikTok"

**Card Interior:**
- Dark background (matching the site's existing dark theme)
- The icon should be large (48-64px equivalent)
- Template name in bold white, large font
- Tagline in a slightly lighter color beneath the name
- The bullet points / features should be scannable — use small icons or emoji prefixes for each
- A "Deploy This Team" or "Get Started" CTA button at the bottom of each featured card
- The optional addon callout sits in a subtle bordered box at the bottom — clearly differentiated from the core features (e.g. lighter background, italic 'optional' tag)

**Section Header:**
- Above the featured cards, add a section header like: "🔥 Featured Templates" or "⚡ Flagship Agent Teams" — something that signals these are the premium, most powerful options
- Optional: A one-liner beneath the header like "Our most advanced agent teams — built for serious operators."

**Regular Templates Grid:**
- The remaining templates (service business, ecommerce, content creator, agency, etc.) should still display in the existing grid format below the featured section
- Remove Ghost Operator from the regular grid since it's now in the featured section
- The TikTok Shop Operator should ONLY appear in the featured section, not duplicated in the grid
